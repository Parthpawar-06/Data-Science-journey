## Concepts Covered
- INNER JOIN syntax and ON condition
- Joining two tables on a foreign key
- Combining JOIN with GROUP BY and aggregate functions

## Key mental model
INNER JOIN = only rows that exist in BOTH tables.
LEFT JOIN = all rows from left table, NULL if no match on right.
Always draw the tables and their relationship before writing the query.
